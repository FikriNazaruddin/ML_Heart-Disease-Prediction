# ML_Heart-Disease-Prediction
Program ini merupakan aplikasi machine learning berbasis Streamlit yang digunakan untuk memprediksi risiko penyakit jantung berdasarkan data klinis pasien. Model utama yang digunakan adalah XGBoost (Extreme Gradient Boosting) yang telah dilatih sebelumnya menggunakan dataset penyakit jantung, kemudian disimpan dalam bentuk pipeline terintegrasi
